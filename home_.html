.ROWS and .COLS

.ROWS & .COLS are our flex containers. They both fit all the available space (of their direct parent) and enables a flex context for all their direct children (.ROW or .COL).

.ROW-FLEX / .COL-FLEX

With .COLS and .COL-FLEX we can split, in equal parts all the horizontal available space with columns of the same height (all the available height). With .ROWS and .ROW-FLEX we can split, in equals parts all the vertical space with rows of the same width (all the available width).

Code example: 
<div class="EXTENDER">
    <div class="PADDER25">
       <div class="COLS">
            <divclass="COL-FLEX">a</div>
            <div class="COL-FLEX">a</div>
           <div class="COL-FLEX">a</div>
       </div>
    </div>
</div>

Obviously we can do the same with .ROWs
and .ROW-FLEX

What is really cool is combine flex element with fixed size elements. This is probably the main target of this css library and the goal we actually want to reach in our web applications. Almost all components in a web application can be arranged (at high level) by using a rows and cols structure. But as you know, when we thinking at the content of a single row (col) often we want to put some specific elements, with specific sizes, and have a flexible element the fit the remaining space. A classical example is a fixed width sidebar on the left and the remaining viewport space that can extends and fit all the available space.

Code example: 
<div class="EXTENDER">
    <div class="PADDER25">
       <div class="COLS">
            <divclass="COL" style="width: 80px; ">sidebar</div>
           <div class="COL-FLEX">a</div>
       </div>
    </div>
</div>

Combine .ROWS and .COLS

Lets see what we can get if we combine rows and cols.

Code example: 
<div class="EXTENDER">
    <div class="PADDER25">
       <div class="ROWS">
            <divclass="ROW" style="height: 25px; ">header</div>
           <div class="ROW-FLEX COLS">
               <div class="COL" style="width: 80px;">sidebar</div>
               <div class="COL-FLEX">remaining space</div>
           </div>
            <divclass="ROW" style="height: 25px; ">footer</div>
        </div>
   </div>
</div>

I don't know what are you looking for and why you landed on this page but personally I think this is the real holy grail in web application design. There are a lot of solutions out there in the net that uses css hacks, workarounds, invasive javascript and so on. This Flexible Box Layout Module based solution allows you to use css flex properties to built web layout scaffolding in seconds. All the previous css classes lets you think a layout and write it at the same time by using a based table approach, a super easy and super familiar way to think about a layout. You don't have to study all the flex-box CSS properties cause the .EXTENDER, .PADDER, .ROWS and .COLS will do the dirty work for you.

.EXTENDER

This is the main container used as the root tag of each component. It's a position:absolute tag that fit all the available space (in horizontal and vertical). Its direct children must to be a .PADDER or a .ROWS or a .COLS.

Code example: 
<div class="EXTENDER" style="background-color: rgba(0,0,255,0.2)">
    this is an extender
</div>
Code in action:
this is an .EXTENDER

.PADDER

This tag sets an uniform padding inside an .EXTENDER. By default you can set a padding of 5, 10, 15, 25, 35, 45, 55 or centered.

Code example: 
<div class="EXTENDER" style="background-color: rgba(0,0,255,0.2)">
    <div class="PADDER25" style="background-color: rgba(120,0,255,0.2)">
        this is a .PADDER25
    </div>
</div>
Code in action:
this is a .PADDER25

A numbered .PADDER will fit all the available space inside the .EXTENDER but leaving a frame. No matter the sizes, a .PADDER5 will fit all the available space inside an .EXTENDER leaving a 5 pixel frame all around.

But many times we need to positioning a vertical and horizontal centered object inside a container. In this case we can use .PADDER-CENTER. Now the content inside the .PADDER-CENTER will centered inside the .EXTENDER.

Code example: 
<div class="EXTENDER" style="background-color: rgba(0,0,255,0.2)">
    <div class="PADDER-CENTER" style="background-color: rgba(120,0,255,0.2)">
        <div style="width: 200px; height: 30px; border: solid 1px white">
            this is a .PADDER-CENTER
        </div>
    </div>
</div>

.ROWS and .COLS

.ROWS and .COLS are our flex containers. They both fit all the available space (of their direct parent) and enables a flex context for all their direct children (.ROW or .COL).

.ROW and .ROW-FLEX / .COL and .COL-FLEX

With .COLS and .COL-FLEX we can split, in equal parts all the horizontal available space with columns of the same height (all the available height). With .ROWS and .ROW-FLEX we can split, in equals parts all the vertical space with rows of the same width (all the available width).

Code example: 
<div class="EXTENDER" style="background-color: rgba(0,0,255,0.2)">
    <div class="PADDER25" style="background-color: rgba(120,0,255,0.2)">
        <div class="COLS">
            <div class="COL-FLEX">a</div>
            <div class="COL-FLEX">a</div>
            <div class="COL-FLEX">a</div>
        </div>
    </div>
</div>

Obviously we can do the same with .ROWS and .ROW-FLEX

What is really cool is combine flex element with fixed size elements. This is probably the main target of this css library and the goal we actually want to reach in our web applications. Almost all components in a web application can be arranged (at high level) by using a rows and cols structure. But as you know, when we thinking at the content of a single row (col) often we want to put some specific elements, with specific sizes, and have a flexible element the fit the remaining space. A classical example is a fixed width sidebar on the left and the remaining viewport space that can extends and fit all the available space.

Code example: 
<div class="EXTENDER" style="background-color: rgba(0,0,255,0.2)">
    <div class="PADDER25" style="background-color: rgba(120,0,255,0.2)">
        <div class="COLS">
            <div class="COL" style="width: 80px; background-color: rgba(0,0,0,0.05)">sidebar</div>
            <div class="COL-FLEX">a</div>
        </div>
    </div>
</div>

        </div>

Combine .ROWS and .COLS

Lets see what we can get if we combine rows and cols.

Code example: 
<div class="EXTENDER" style="background-color: rgba(0,0,255,0.2)">
    <div class="PADDER25" style="background-color: rgba(120,0,255,0.2)">
        <div class="ROWS">
            <div class="ROW" style="height: 25px; background-color: rgba(0,0,0,0.05)">header</div>
            <div class="ROW-FLEX COLS" style="resize: none">
                <div class="COL" style="width: 80px; background-color: rgba(0,0,0,0.1)">sidebar</div>
                <div class="COL-FLEX">remaining space</div>
            </div>
            <div class="ROW" style="height: 25px; background-color: rgba(0,0,0,0.05)">footer</div>
        </div>
    </div>
</div>

I don't know what are you looking for and why you landed on this page but personally I think this is the real holy grail in web application design. There are a lot of solutions out there in the net that uses css hacks, workarounds, invasive javascript and so on. This Flexible Box Layout Module based solution allows you to use css flex properties to built web layout scaffolding in seconds. All the previous css classes lets you think a layout and write it at the same time by using a based table approach, a super easy and super familiar way to think about a layout. You don't have to study all the flex-box CSS properties cause the .EXPANDER, .PADDER, .ROWS and .COLS will do the dirty work for you.
